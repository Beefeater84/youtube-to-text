# Страница dashboard
## Права доступа
- Только авторизованные пользователи могут видеть эту страницу.

### Функционал и блоки

### Форма для создания новой задачи
- Она принимает на вход YouTube URL
- Список доступных языков для перевода Youtube видео
- Клиент выбирает любой язык из доступных, и отправляет форму. 
- Английский язык выбран по умолчанию. Клиент может его переключить (снять/поставить), но при отправке формы EN включается всегда, независимо от выбранных языков.

**Use case 1:**

- Клиент выбирает видео
- Выбирает английский

Ожидаемое поведение:

- В базе данных появляется запись с языком en

**Use case 2:**

- Клиент выбирает видео
- Выбирает русский

Ожидаемое поведение:

- В базе данных появляется запись с языком ru
- В базе данных появляется запись с языком en


### Список видео клиента.

- Список видео, которые были созданы пользователем
- Если видео представлены на разных языках, то выводится только одно видео с показом языков, которые есть в базе данных.

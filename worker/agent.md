# Worker agent

Связаннок покимание. Это функция API отправляет задачи из web в базу данных [web\features\create-transcript\api\agent.md]

Worker каждые 5 секунд проверяет базу данных на наличие задач в статусе "pending".

## Use cases

### Use Case 1:

- В видео нет никаких субтитров 

Ожидаемое поведение:

- Записать в базу данных статус "error"
- Записать в базу данных ошибку "No subtitles found"
- Вернуть ошибку "No subtitles found"

### Use Case 2:

- Пришло видео на перевод на НЕ английский язык
- У нас в базе уже есть запись с таким видео на английском языке

Ожидаемое поведение:
У нас в задаче всегда ожидается что один из языков будет английский

- Проверить есть ли такое видео на английском языке в базе данных
- Взять md файл с английским переводом - отправить на перевод
- Сохранить md файл с переводом на НЕ английский язык в базу данных
- Записать в базу данных статус "done"
- Вернуть success: true

### Use Case 3:

- Пришло видео на перевод на НЕ английский язык
- У нас в базе нет записи с таким видео на английском языке

Ожидаемое поведение:

- Создать в базе данных запись с таким видео на английском языке
- Приоритезировать задачу для перевода с этим видео на английском языке перед другими задачами
- Текущей задаче присвоить статус "pending"
- Вернуть success: true

### Use Case 4:

- Пришло видео на перевод и у Видео нет английских субтитров

Ожидаемое поведение:

- Определить какой язык у видео
- Взять субтитры - отправить на перевод на английский язык
- Дальше стандартный пайплайн: чистка и структуризация с помощью OpenAI, сохранение в md, публикация на сайте
